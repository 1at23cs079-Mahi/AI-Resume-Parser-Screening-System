# AI Resume Parser & Screening System

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2F1at23cs079-Mahi%2FAI-Resume-Parser-Screening-System)

AI Resume Parser & Screening System is an intelligent recruitment platform that automates the resume screening process using Artificial Intelligence, Machine Learning, and Natural Language Processing (NLP). The system helps recruiters analyze candidate resumes efficiently and identify the most suitable applicants for a job role.

The application extracts important information such as skills, technologies, education, and experience from resumes and evaluates candidates based on contextual relevance instead of simple keyword matching. This approach helps reduce keyword stuffing and improves the accuracy of candidate shortlisting.

Using NLP preprocessing techniques with Python and the NLTK library, the system analyzes resume content and compares it with job-related requirements. Machine Learning techniques are used to identify relevant skills and generate an ATS-style resume score ranging from 0 to 10 based on candidate suitability.

## Features

* Automated resume parsing and analysis
* AI-powered candidate screening
* NLP-based skill extraction
* Context-aware skill matching
* ATS-style resume scoring
* Detection of irrelevant keyword stuffing
* Fast and efficient candidate shortlisting

## Technologies Used

* Python
* NLTK
* Scikit-learn
* Machine Learning
* Natural Language Processing (NLP)

## Dataset

The project uses the Kaggle Resume Dataset containing job titles and resume text data for training and evaluation purposes.

## Objective

The main objective of this project is to simplify the recruitment process by automating resume analysis and helping recruiters identify qualified candidates quickly and accurately using AI-powered techniques.
